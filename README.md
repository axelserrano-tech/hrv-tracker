Kubios-Style HRV Readiness Tracker: A professional-grade cardiovascular health dashboard designed for students and athletes to track recovery using Heart Rate (HR) and Heart Rate Variability (HRV).
Project Overview: This application provides a downloadable, OS-agnostic solution for collecting valid and reliable physiological measurements. It establishes a personal baseline for each user and provides intuitive feedback via a "Kubios-style" readiness gauge.
Key Features: Scientific Baselines: Automatically computes the mean HR and HRV (RMSSD) from repeated measurements.Readiness Speedometer: Uses color-coded zones based on daily deviation from the individual's norm.Subjective Wellness: Includes a 1-10 soreness scale and an anatomical body map for marking pain locations.Coach/Admin View: A centralized dashboard for administrators to monitor team compliance and group trends with full timestamp visibility.
Understanding the Readiness GaugeThe gauge uses Standard Deviation (Z-Scores) to determine your daily status:
READY🟢Green-Within 0.5 SD of baseline. Optimal for high-intensity training.
CAUTION🟡Yellow-Between 0.5 and 1.5 SD below baseline. Consider active recovery.
REST🔴Red->1.5 SD below baseline. High cardiovascular strain; rest is recommended.
Biological Considerations: This app accounts for physiological fluctuations as requested:Resting HR: Normative values are set between 60-100 BPM.Bio-Factors: The app records Weight and Sex, acknowledging that heart size, hormones, and tissue supply needs (higher body weight) naturally influence resting HR.
